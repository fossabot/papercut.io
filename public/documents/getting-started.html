Getting Started with Papercut

Papercut is a desktop application for full-text search across your HTML document collection. It indexes your documents at build time and provides instant, offline search capabilities.

Key Features

  • Lightning-fast full-text search powered by Pagefind
  • Completely offline — no internet connection required
  • Built with Tauri for a lightweight desktop experience
  • Search across thousands of HTML documents instantly

How It Works

Documents are indexed during the build process using Pagefind, which creates a compressed search index. At runtime, only the relevant portions of the index are loaded into memory, keeping the application fast and responsive even with large document collections.

Adding Your Documents

Place your HTML files in the documents/ directory. Each document should have a proper HTML structure with a title tag and heading elements for best search results. Pagefind will automatically extract and index the text content.
