Search Tips and Techniques

Get the most out of Papercut's search functionality with these tips.

Basic Search

Simply type a word or phrase into the search bar. Papercut will return documents containing your search terms, ranked by relevance. Results appear as you type with no delay.

Phrase Search

Use quotation marks to search for an exact phrase. For example, searching for "full text search" will only return documents containing that exact sequence of words.

Understanding Results

Each search result displays the document title and a text excerpt showing where your search terms appear. Matched terms are highlighted for easy identification. Click any result to view the full document content.

Performance

Papercut uses a chunked index system. This means the search engine only loads the portions of the index relevant to your query, resulting in fast search times even across thousands of documents. The initial search may take slightly longer as index chunks are loaded, but subsequent searches are nearly instantaneous.
