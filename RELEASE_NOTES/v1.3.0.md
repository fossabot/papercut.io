# Papercut v1.3.0 - Release Notes

- **Target release:** 1.3.0
- **Baseline:** [`v1.2.0`](https://github.com/muhannadnouri/papercut.io/releases/tag/v1.2.0)
- **Release branch:** `branch-release-v1.3.0`
- **Primary change:** first official signed and notarized macOS release artifacts.

Papercut v1.3.0 is a focused packaging release. App behavior is intentionally the same as v1.2.0; the release makes macOS a supported GitHub Release download alongside Linux, Windows, and Android.

## Highlights

- **macOS release artifacts are now official.** The release workflow builds per-architecture DMGs for Apple Silicon and Intel Macs.
- **Developer ID signing and notarization.** macOS release jobs import the Developer ID Application certificate from the protected `apple-release` GitHub Environment, sign with hardened runtime entitlements, notarize the app bundle, notarize/staple the outer DMG, and verify with `codesign`, `stapler`, and `spctl` before upload.
- **Cleaner release documentation and site downloads.** Public install notes now distinguish official signed/notarized release DMGs from unsigned local or PR development artifacts.

## Test Tag Note

Tags `v1.2.1` through `v1.2.6` were release-pipeline validation tags used while bringing up macOS Developer ID signing, notarization, DMG stapling, and Gatekeeper verification. Treat them as CI/release-infrastructure test tags, not product releases. The next user-facing release after `v1.2.0` is `v1.3.0`.

## Compatibility

No saved-data or audiobook cache migration is expected. The native audiobook cache version remains unchanged. Users upgrading from v1.2.0 should see the same app behavior, with macOS users now able to install from signed/notarized DMGs.

## Verification

- GitHub release workflow builds Linux, Windows, Android, and macOS artifacts.
- macOS Apple Silicon and Intel jobs complete in the protected `apple-release` environment.
- macOS app bundle passes `codesign --verify` and Gatekeeper execute assessment.
- macOS DMG artifacts are separately notarized, stapled, validated, and pass Gatekeeper open assessment.

## Superseded macOS Artifact Note

After v1.3.0 validation, the macOS DMGs were found to pass signing/notarization while missing required native TTS dylibs in `Papercut.app/Contents/Resources`, causing launch failure on macOS. Use v1.3.3 or newer for macOS installs.
