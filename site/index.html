Offline document companion

Search, Read, Listen Offline

Papercut is a lightweight desktop and Android reader for local documents, full-text search, saved audiobook playback, and a private offline library. No accounts. Reading and search work offline; TTS models download once if you use audio.

Features

Offline search

Full-text search across every document in your library. Type and find without internet, accounts, or waiting. Imported HTML and EPUB files join the same search flow.

Native TTS audiobooks

Turn any document into an audiobook with natural-sounding offline text-to-speech. English and Arabic voices available. Download a voice once, then listen anywhere, including background and lock-screen playback on Android.

Bring your own documents

Import your own HTML and EPUB files, organize them into folders, tune the reader with fonts and dark mode, and convert supported documents to audio. Works on Windows, macOS, Linux, and Android. PDF support coming soon.

Install FAQ

Common install questions for each platform. On every platform, TTS voice models are downloaded only when you choose to use audio.

Android

Enable Install unknown apps in Settings for your browser or file manager, then open the APK. Reading and search work offline. Network access is only used if you choose to download a TTS model; if you do not use TTS, network access is not needed.

Linux

Debian-based (Ubuntu, Mint): sudo apt install ./Papercut_*.deb

RPM-based (Fedora, openSUSE): sudo rpm -i Papercut-*.rpm

AppImage: make executable (chmod +x) and run directly. On Arch-based systems the AppImage may need WEBKIT_DISABLE_COMPOSITING_MODE=1; see the README for details.

macOS

Open the .dmg and drag Papercut to your Applications folder. Official release builds are Developer ID signed and notarized. If you are testing an unsigned CI or local development artifact, right-click (or Control-click) the app icon and choose Open on first launch.

Pick the build that matches your Mac: Apple Silicon (_aarch64.dmg) for M-series chips, or Intel (_x64.dmg) for Intel-based Macs.

Windows

The MSI installer handles setup automatically. If you prefer the portable EXE, run it directly. No admin rights needed.
